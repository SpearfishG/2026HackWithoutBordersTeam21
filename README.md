# 2026HackWithoutBordersTeam21
The codes that our team used to build our solution, JavaScript, HTML, and CSS. 

## Features: 
This identified keywords on a webpage, 
connected to an external AI which searched through its database to provide contextual results, 
and displayed those results on user's end upon prompt.
